# AI Agent Conversation ID Tracing

This example demonstrates how to implement observability in a simplified AI agent using unique conversation IDs. It simulates a user interaction that goes through multiple internal steps, with each step logging its activity alongside a consistent conversation ID. This allows for easy tracing of a single user's request across the agent's entire workflow, crucial for debugging and performance monitoring.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal: `python main.py`
